# Google Ads Integration

Google Ads integration is coming soon to Dotoro!

## Planned Features

### Campaign Types

We're building support for:

- **Search Ads** - Text ads on Google Search
- **Display Ads** - Visual ads across the Display Network
- **Performance Max** - AI-powered cross-channel campaigns
- **Shopping Ads** - Product-based advertising

### Targeting Options

Planned targeting capabilities:

- Keywords and search terms
- Audience segments
- Demographics and locations
- Remarketing lists

### Bid Strategies

Support for Google's smart bidding:

- Target CPA
- Target ROAS
- Maximize conversions
- Maximize conversion value

## Current Alternatives

While waiting for native integration:

### API Access

Use Dotoro's API to build custom integrations:

```bash
# Export campaign data for Google Ads
curl https://api.dotoro.io/v1/campaign-sets/{id}/export \
  -H "Authorization: Bearer $API_KEY" \
  -o campaigns.csv
```

### Manual Export

1. Generate campaigns in Dotoro
2. Export to CSV
3. Import via Google Ads Editor
