# Meta Ads Integration

Meta Ads integration for Facebook and Instagram is coming soon to Dotoro!

## Planned Features

### Campaign Types

We're building support for:

- **Image Ads** - Single image creatives
- **Video Ads** - Video content ads
- **Carousel Ads** - Multi-image swipeable ads
- **Collection Ads** - Product catalog ads
- **Lead Ads** - In-app lead generation

### Placements

Planned placement support:

- Facebook Feed
- Instagram Feed
- Facebook Stories
- Instagram Stories
- Messenger
- Audience Network

### Targeting Options

Planned targeting capabilities:

- Core audiences (demographics, interests, behaviors)
- Custom audiences (website, app, customer lists)
- Lookalike audiences
- Advantage+ targeting

### Objectives

Support for Meta's campaign objectives:

- Awareness
- Traffic
- Engagement
- Leads
- Sales
- App promotion

## Timeline

We're actively developing Meta Ads integration. Key milestones:

| Phase 2 | Video, Carousel, Stories | Planned |

## Current Alternatives

While waiting for native integration:

### API Access

Use Dotoro's API to build custom integrations:

```bash
# Export campaign data for Meta Ads
curl https://api.dotoro.io/v1/campaign-sets/{id}/export?format=meta \
  -H "Authorization: Bearer $API_KEY" \
  -o campaigns.json
```

### Manual Export

1. Generate campaigns in Dotoro
2. Export campaign data
3. Create campaigns in Meta Ads Manager

### Creative Export

Export creative assets for Meta:

1. Download generated ad copy
2. Download associated images/videos
3. Upload to Meta's Asset Library
