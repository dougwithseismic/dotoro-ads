## Step 2: Create Platform-Specific Templates

### Reddit Template

```yaml
name: Reddit - Product Ads
platform: reddit

campaign:
  name: "[Reddit] \${category} Products"
  objective: conversions
  budget: \${reddit_budget}

ad_group:
  name: "\${name}"
  targeting:
    communities: \${reddit_subreddits}
  bid:
    strategy: cpc
    amount: 0.75

ad:
  headline: "\${reddit_headline|\${name} - Now $\${price}}"
  body: |
    \${description}

    Check it out: \${url}
  image: \${image_url}
```

**Reddit-specific considerations:**
- Conversational tone
- Community-aware targeting
- Longer body text allowed

### Google Template

```yaml
name: Google - Search Ads
platform: google

campaign:
  name: "[Google] \${category}"
  objective: conversions
  budget: \${google_budget}

ad_group:
  name: "\${name}"
  keywords: \${google_keywords}
  match_type: phrase

ad:
  headlines:
    - "\${google_headline_1|\${name}}"
    - "\${google_headline_2|$\${price}}"
    - "Shop \${category} Today"
  descriptions:
    - "\${description}"
    - "Free shipping on orders $50+"
  final_url: \${url}?utm_source=google
```

**Google-specific considerations:**
- Multiple headline variations
- Character limits (30 chars headlines)
- Keyword relevance

### Meta Template

```yaml
name: Meta - Image Ads
platform: meta

campaign:
  name: "[Meta] \${category}"
  objective: conversions
  budget: \${meta_budget}

ad_set:
  name: "\${name}"
  targeting:
    interests: \${meta_interests}
    age_range: [25, 55]
    placements: [facebook_feed, instagram_feed]

ad:
  primary_text: "\${meta_primary_text|\${description}}"
  headline: "\${name}"
  description: "Starting at $\${price}"
  link: \${url}?utm_source=meta
  image: \${image_url}
  cta: "Shop Now"
```

**Meta-specific considerations:**
- Visual-first format
- Detailed audience targeting
- Emoji-friendly copy

## Step 3: Coordinate Budgets

### Budget Allocation Strategy

Allocate your total budget across platforms:

**Option 1: Fixed Split**
```csv
product_id,total_budget,reddit_budget,google_budget,meta_budget
SKU-001,100,20,50,30
SKU-002,150,30,75,45
```

**Option 2: Calculated Split**
Add a transform to calculate platform budgets:

```yaml
transform:
  name: Budget Calculator
  compute:
    reddit_budget: total_budget * 0.20
    google_budget: total_budget * 0.50
    meta_budget: total_budget * 0.30
```

**Option 3: Performance-Based**
Use rules to adjust based on historical performance:

```yaml
rules:
  - name: Boost performing platforms
    condition: google_roas > 3
    action: google_budget = google_budget * 1.2
```

### Performance Comparison

Compare the same products across platforms:

```sql
-- Example analysis query
SELECT
  product_id,
  platform,
  SUM(spend) as spend,
  SUM(conversions) as conversions,
  SUM(revenue) / SUM(spend) as roas
FROM campaign_performance
GROUP BY product_id, platform
ORDER BY roas DESC
```

## Optimization Strategies

### Platform Rebalancing

Shift budget to top performers:

```yaml
rule: Weekly Platform Rebalancing
trigger: weekly
action:
  - Calculate ROAS per platform
  - Increase budget 10% for top performer
  - Decrease budget 10% for lowest performer
  - Maintain minimum $500/platform
```

### Message Testing

Test different messaging per platform:

| Version | Reddit | Google | Meta |
|---------|--------|--------|------|
| A | Community tone | Feature-focused | Lifestyle |
| B | Technical specs | Benefit-focused | Social proof |
| C | Reddit memes | Problem-solution | User testimonials |

### Audience Expansion

Use insights to expand targeting:

1. Identify top-performing audiences per platform
2. Create lookalike audiences (Meta)
3. Expand keyword lists (Google)
4. Target related subreddits (Reddit)

## Troubleshooting

### "Platform sync out of date"

1. Check individual platform connection status
2. Verify API credentials haven't expired
3. Review platform-specific error logs

### "Budget overspend"

1. Verify platform-level caps are set
2. Check for overlapping audience (cross-platform attribution)
3. Adjust frequency caps

### "Inconsistent performance data"

1. Confirm attribution windows match across platforms
2. Account for platform reporting delays
3. Use Dotoro's unified reporting for consistency

## Best Practices

### Creative Consistency

1. **Unified brand voice** - Adapt tone, maintain identity
2. **Consistent visuals** - Same product images, platform-optimized sizes
3. **Aligned messaging** - Core value prop across platforms

### Operational Efficiency

1. **Single source of truth** - One data source, multiple outputs
2. **Template inheritance** - Base template with platform overrides
3. **Centralized rules** - Global rules applied across platforms

### Performance Focus

1. **Platform-appropriate KPIs** - Different goals per platform
2. **Cross-platform attribution** - Understand the full journey
3. **Incremental testing** - Test one variable at a time
