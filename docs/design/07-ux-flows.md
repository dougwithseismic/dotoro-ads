### Step Breakdown

| Step | Name | Required | Description |
|------|------|----------|-------------|
| 1 | Data Source | Yes | Select CSV or API data source |
| 2 | Rules | No | Filter/transform data before generation |
| 3 | Campaign Config | Yes | Campaign name pattern, objective |
| 4 | Ad Type | Yes | Select ad format(s) per platform |
| 5 | Ad Structure | Yes | Configure ads, creatives, hierarchy |
| 6 | Budget & Bidding | Yes* | Budget, bidding strategy, schedule |
| 7 | Targeting | No | Location, audience, interests |
| 8 | Preview | Yes | Review and generate campaigns |

*Budget can use defaults or be skipped for draft campaigns

## Responsive Considerations

### Desktop (1200px+)

- Full side panel visible
- All sections expanded by default
- Multi-column layouts where appropriate

### Tablet (768px - 1199px)

- Side panel collapsible
- Reduced padding
- Single column for complex forms

### Mobile (< 768px)

- Side panel hidden (accessible via button)
- Single column layout
- Accordion for long sections
- Bottom sheet for previews
